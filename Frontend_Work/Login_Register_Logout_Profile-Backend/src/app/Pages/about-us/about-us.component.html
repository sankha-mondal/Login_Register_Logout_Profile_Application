About Us!

This is a simple hero unit, a simple Angular Project for uploading files like .pdf, .doc, .exe, .ppt, .png, .jpg, .jpe, .mp3, mp4, .zip, .html, .css, .ts, .js, .java, .py, .sql etc upto 4 mb. This files are stored into file_db database. We can Download and Delete all the respective files with respected to File-Id. This project has only one table named "files" having 'id' i.e PK, name, type, data are the fields. In this project for Spring-Boot 'date' is of 'byte[]' type, for MySQL 'date' is of 'longblob' type, for Angular 'date' is of 'any' type.

It uses utility classes for typography and spacing to space content out within the larger container.
